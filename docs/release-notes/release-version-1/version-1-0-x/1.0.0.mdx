*Release date: 09 March 2026*

WaveMaker announces the release of WaveMaker AI 1.0.0 Beta. This release is intended for early adopters and developers who want to explore the capabilities of the WaveMaker platform.

For details about the technology stack upgrades, refer to [Technology Stack](/tech-stack).

    <Accordian title="User Interface">
      - ### Design System based projects <Pill type="web" /> <Pill type="mobile" />

        [Documentation](../../../design-system/concepts)

      - ### Auto Layout for responsive UIs <Pill type="web" /> <Pill type="mobile" />

        Create structured, responsive layouts visually with precise control over alignment, spacing, orientation and sizing—without writing CSS.

        [Documentation](../../../user-interfaces/web/develop/working-with-layouts/auto-layout)

      - ### Autocode for converting Design-to-code <Pill type="web" />

        Convert supported Figma designs into working WaveMaker applications with layouts, components, and auto-generated tokens.

        [Documentation](../../../design-system/figma-autocode-plugin/working-with-autocode-plugin.md)

      - ### Web Applications <Pill type="web" />
        - **Angular (Default)**: Build robust enterprise grade web applications with our optimized Angular-based output.
        - **React (Beta)**: **React as a build output for Web**. This beta feature allows you to export production-ready React code while using the same visual development experience.

      - ### Mobile Applications <Pill type="mobile" />
        - **React Native (Default)**: Build and deploy high-performance, native mobile applications for iOS and Android using **React Native** as the primary target framework.
    </Accordian>
